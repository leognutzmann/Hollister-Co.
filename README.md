## Getting Started

To run this project you must have Expo in your mobile device or an emulator to preview the app. You can find more information here https://docs.expo.io/versions/latest/


### Prerequisites

You must have NPM and Node.js installed. To install run 
```sh
curl -sL https://deb.nodesource.com/setup_10.x | sudo -E bash -
sudo apt install nodejs
```

To update your npm run
```sh
npm install npm@latest -g
```
## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

See [Configuration Reference](https://cli.vuejs.org/config/).
